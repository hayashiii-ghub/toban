モーダルズレの原因分析:

問題: 画面の拡大率（ブラウザズーム）によって設定モーダルが背景要素とズレる
スクリーンショット: モーダルが画面中央に表示されず、背景のコントロールバーやボタンと重なっている

原因:
1. モーダルが `<div className="min-h-screen">` の中に配置されている（370行目付近）
   → fixed positioning は本来ビューポート基準だが、親要素に transform がある場合は
     その要素が新しい containing block になる
2. framer-motion の AnimatePresence/motion.div が transform を使用するため、
   fixed の基準がビューポートではなく親要素になっている可能性
3. ブラウザズームで transform の計算がずれる

修正方針:
- モーダルを React Portal で body 直下にレンダリングする
  → これにより親要素の transform の影響を受けなくなる
- または、モーダルのオーバーレイを min-h-screen の div の外に移動する
